We appreciate your help for extending the dataset. You can upload sketch files of different eyebrows, eyes, nose, and mouths. We will combine these hand-drawn facial features into different faces.

Please upload the file as following steps,

1 File Format

Please save the sketch as svg file. If you want to upload more than one sketches once, we strongly suggest you compress them as one file and then upload it.

2 File Name

Each sketch should be named as "Facial feature _ id(Gender + Viewing angle + Emotion + Style)", for example, "eyes_01(male + frontal + happy + cartoon).svg"

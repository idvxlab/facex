A Vector Drawing Dataset Containing 5,240,088 Face Sketches

The dataset has different categories, including genders (male, female), viewing angles (frontal, mid-profile left, mid-profile right view), emotions (neutral, happy, sad, angry, fearful, surprised, disgusted), and artistic styles (realistic, cartoon, abstract styles).

Construct the FaceX Dataset

We invited a group of designers to draw the dataset.

1 Draw the Facial Features

First, the designers drew faical features with neutral emotion in front view. Then, they drew the facial features with six basic emotions following some rules based on a literature survey. After that, they created those features in mid-profile left view.

2 Adjust Each Line

Designers adjusted the anchor points in each line to ensure the two handles of the point in line so that the lines in each sketch are smooth and neat.

3 Generate Faces

We placed the facial features that designers drew based on the divine proportion of the human head and combined these hand-drawn facial features into different faces.

We provide two data formats of FaceX.

Svg File

Scalable Vector Graphics. An XML-based vector image format for two-dimensional graphics with support for interactivity and animation.
Learn more.

Npz File

A sequence of stroke vectors, including three stroke states: a stroke continuous from the last point, a stroke ends, and a drawing ends.
